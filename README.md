# 🐍 Estudos de Programação Orientada a Objetos (POO) em Python

Este repositório foi criado para consolidar meus aprendizados sobre **Programação Orientada a Objetos (POO)** utilizando Python. Aqui você encontrará os desafios propostos no [Curso de Python - Mundo 4](https://www.cursoemvideo.com/curso/python-3-mundo-4-40-horas/aulas/mundo-4/modulos/aprenda-programacao-orientada-a-objetos-com-desafios-em-python/) do Curso em Vídeo, além de projetos autorais criados para fixação dos conceitos.

## 🚀 Conceitos Praticados

Durante os desafios e treinos, explorei os pilares e conceitos fundamentais de POO:
*   **Abstração:** Criação de classes representando entidades do mundo real.
*   **Encapsulamento:** Proteção de atributos e métodos (uso de `getters`, `setters` e propriedades).
*   **Herança:** Reutilização de código e especialização de classes.
*   **Polimorfismo:** Sobrescrita de métodos para comportamentos específicos.
